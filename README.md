# GoCQLSockets
Communicate with Cassandra using GoCQL and websockets

# Install Cassandra
http://cassandra.apache.org/download/

## Run Cassandra
%\apache-cassandra\bin\cassandra

# Run sockets
 
## Setup config
Edit conf.json to your setup 

## Run client
go run main.go --mode client

## Run server
go run main.go --mode server
